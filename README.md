
# Clarity Tools

Clarity is a multi tool for OSINT ip lookup web poage copier and more


## Features

- [x] OSINT
- [x] Discord token info
- [x] Whois lookup
- [X] Ip lookup
- [ ] Web page copier
- [ ] ClarityAI
- [x] Username tracker
- [ ] doxbin tracker
- [x] discord server info



## Deployment

To deploy this project run

```bash
  pip install -r requierements.txt
```

for run

```bash
  python main.py
```
